# Chat 页面设计文档

## 路由设计
- 页面路由: `/chat`
- API 路由: 
  - `/api/chat/send` - 发送消息
  - `/api/chat/history` - 获取聊天历史
  - `/api/articles/search` - 搜索文章（用于左侧列表）
  - `/api/articles/<article_id>` - 获取文章详情
